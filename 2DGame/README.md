# Clic2DUnityGame
Unity 2D Game Repository

# Naming Conventions
Assets (art, PNG files, etc) Files, Scripting Files, Functions, GameObjects, Folders: Capitalize the first letter of each word</br>
Variables: camelCase

# Assets Sizes
Characters: min 64x64 pixels
